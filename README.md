# Sudoku solver [WIP]

![demo](./images/demo.gif)

## Features (right now)

Right now everything is terminal based and in a very simple way displayed there

- Display a sudoku
- Create a new sudoku
- Move in grid
  - Move with multiplicators i.e. 7nc would move 7 columns to the right
- Write digits
- Remove digits
- Quit program
